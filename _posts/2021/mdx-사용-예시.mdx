## React Live Editor
react 코드를 바로 실행할 수 있는 기능이다. '```jsx react-live'로 코드 블럭을 감싸면 된다.
```jsx react-live
const handleClick = () => { alert("Hello!") }
render(<button onClick={handleClick}>Click</button>)

```
