# 기존 markdown 문법
## Headings

```md
# H1 Heading

## H2 Heading

### H3 Heading

```

실제 표현
# H1 Heading

## H2 Heading

### H3 Heading

--- 

## Texts

```md
Here's some basic text

Here's some _italics_

Here's some **bold** text

Here's some **_bold italics_** text

Here's some ~~strikethrough~~ text

```

실제 표현

Here's some basic text

Here's some _italics_

Here's some **bold** text

Here's some **_bold italics_** text

Here's some ~~strikethrough~~ text

---

## Link

```md
[Click here](https://google.com)

```

실제 표현

[Click here](https://google.com)

---

## Image


![](../images/somePhoto.jpeg)

## Code block (highlight)

highlight은 주석으로 할 수 있으며 범위 지정은 '// highlight-start'부터 '// highlight-end'까지이며 라인 지정은 '// highlight-line'이다.

실제 표현
```jsx
// highlight-start
import React, { Component } from "react"
class Profile extends Component {
// highlight-end
  render() {
    return <div></div> // highlight-line
  }
}
export default Profile

```

## Code block with title 
code block에 제목을 작성할 경우 ```cpp:title=제목의 형태로 작성하면 된다.

실제 표현
```cpp:title=example.cpp
const int number = 1234;

```

## Diff 
diff의 경우 ```diff로 시작하며 본문에 +, -로 표현해 주면 다른 컬러로 설정된다.

실제 표현
```diff
+ {
+ import React, { Component } from "react"
+
+ class SomeClass extends Component {
-  render() {
-    return (<div></div>)
-   }
+ }
+ export default SomeClass
```
